页眉、页脚中的文档进度条，章进度条。

底部的彩色进度条可以点击跳转至相应章节的开头。
底部的点状进度条则是本章的进度，其中实心圆圈的是已经阅读过的，方形是正在阅读的（也就是当前页），空心圆圈是还没有阅读的。实心圆圈和空心圆圈都是可以点击跳转的，可以跳转至对应的页面。

可以方便的应用于页眉、页脚中。颜色、形状、文字都可以修改。
