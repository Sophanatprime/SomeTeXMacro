# SomeTeXMacro
Some useful TeX macro for myself.

- [bihua](./bihua)：使用 LaTeX 排版汉字笔画顺序。
- [progress-bar](./progress-bar)：文档进度条、章节进度条。
- [style](./style)：各类样式。
- [tikz-code-example](./tikz-code-example)：排版 TikZ 图像及其代码，支持高亮和分页等。
